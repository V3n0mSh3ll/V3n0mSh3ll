I'm a security-focused developer working at the intersection of **offensive security and AI systems**. I build tools that simulate real-world attacks against LLMs and web infrastructure, then use what I learn to make systems harder to break. 4+ years deep in Python, PHP, and JavaScript across security tooling, ecommerce, and production web apps.
